### Branch
---
``` bash
# create the branch
git branch theName

# check the branches
git branch

#switch the branch
git checkout theBranch

# add new file 

git add fileName
git commit -m 'update'
git push


# merge
git merge TheBranchYouWnnaMerge

// upload to GitHub
git status
git add fileName
git commit -m 'update'
git push
git push --set-upstream origin branchName
```
